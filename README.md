# Web3 Airdrop Automation Basic

This project helps beginners learn how crypto airdrops work and how tokens can be sent to many users at once.

You will learn:
- What an airdrop is
- How to connect a wallet
- How to send tokens to many addresses
- How to track transactions
- How Web3 automation works

## Requirements
- Node.js installed
- MetaMask browser extension
- Basic JavaScript knowledge

## Setup

1. Clone the repo
git clone https://github.com/yourusername/web3-airdrop-automation-basic.git

2. Install packages
npm install

3. Run the app
npm start

4. Open browser
http://localhost:3000

## Project Structure

- index.html = Main webpage
- app.js = Frontend logic
- airdrop.js = Airdrop functions
- web3.js = Blockchain connection
- deploy.js = Contract deploy script
- config.js = Network settings
- server.js = Local server
- package.json = Project info
- utils.js = Helper functions
- .env.example = Environment example

## Features
- Connect wallet
- Add wallet list
- Send airdrop
- Show transaction hash
- Simple batch transfer flow

## Learning Goal
This project explains how token airdrops can be automated using blockchain.
